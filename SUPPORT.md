# Support

Lightning P2P support is handled in public by default so fixes and answers help
future users.

## First Places To Check

- [README.md](README.md) for installation and release links
- [SECURITY.md](SECURITY.md) for ticket, relay, signing, and trust guidance
- [docs/download-trust.md](docs/download-trust.md) for Windows installer warnings
- [docs/android-trust.md](docs/android-trust.md) for Android sideload verification
- [docs/README.md](docs/README.md) for deeper project docs

## Ask A Question

Use GitHub Discussions for setup questions, ideas, comparisons, and usage
questions that are not clearly bugs.

## Report A Bug

Open an issue with:

- Lightning P2P version
- sender and receiver operating systems
- whether the transfer was LAN direct, WAN direct, relay fallback, or unknown
- exact steps to reproduce
- expected behavior
- actual behavior
- redacted logs or screenshots when useful

Do not post private receive tickets, private keys, sensitive filenames, or
personal paths in public issues.

## Security Reports

Do not open a public issue with exploit details. Follow [SECURITY.md](SECURITY.md)
and request a private reporting path if needed.
